nuvoton,npcx-tach
#################

Vendor: :ref:`Nuvoton Technology Corporation <dt_vendor_nuvoton>`

Description
***********

These nodes are "tach" bus nodes.

.. code-block:: none

   Nuvoton, NPCX-Tachometer node

**********
