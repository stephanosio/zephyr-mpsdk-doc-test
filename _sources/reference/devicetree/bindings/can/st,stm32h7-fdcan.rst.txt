st,stm32h7-fdcan
################

Vendor: :ref:`STMicroelectronics <dt_vendor_st>`

Description
***********

These nodes are "can" bus nodes.

.. code-block:: none

   Bosch m_can CAN-FD controller for STM32H7 series
