FXOS8700 Accelerometer/Magnetometer Sensor¶

Overview¶

This sample application shows how to use the FXOS8700 driver. The driver supports FXOS8700 accelerometer/magnetometer and MMA8451Q, MMA8652FC, MMA8653FC accelerometers.

Building and Running¶

This project outputs sensor data to the console. FXOS8700 sensor is present on the NXP FRDM-K64F, NXP FRDM-K22F, NXP FRDM-KW41Z, Hexiwear, and NXP TWR-KE18F boards. Accelerometer only devices are present on the NXP FRDM-KL25Z, BBC MicroBit, and reel board boards. It does not work on QEMU.

Building and Running for FRDM-K64F¶

FRDM-K64F is equipped with FXOS8700CQ accelerometer and magnetometer. Sample can be built and executed for the FRDM-K64F as follows:

west build -b frdm_k64f samples/sensor/fxos8700
west flash

Example building for the FRDM-K64F with motion detection support:

west build -b frdm_k64f samples/sensor/fxos8700 -- -DCONF_FILE="prj.conf overlay-motion.conf"
west flash

Building and Running for FRDM-K22F¶

FRDM-K22F is equipped with FXOS8700CQ accelerometer and magnetometer. Sample can be built and executed for the FRDM-K22F as follows:

west build -b frdm_k22f samples/sensor/fxos8700
west flash

Example building for the FRDM-K22F with motion detection support:

west build -b frdm_k22f samples/sensor/fxos8700 -- -DCONF_FILE="prj.conf overlay-motion.conf"
west flash

Building and Running for TWR-KE18F¶

TWR-KE18F is equipped with FXOS8700CQ accelerometer and magnetometer. The FXOS8700CQ IRQ lines, however, are not connected by default, so motion detection is not supported.

Sample can be built and executed for the TWR-KE18F as follows:

west build -b twr_ke18f samples/sensor/fxos8700
west flash

Building and Running for FRDM-KL25Z¶

FRDM-KL25Z is equipped with MMA8451Q accelerometer. Sample can be built and executed for the FRDM-KL25Z as follows:

west build -b frdm_kl25z samples/sensor/fxos8700 -- -DCONF_FILE="prj_accel.conf"
west flash

Building and Running for Micro Bit¶

Micro Bit is equipped with MMA8653FC accelerometer. Sample can be built and executed for the Micro Bit as follows:

west build -b bbc_microbit samples/sensor/fxos8700 -- -DCONF_FILE="prj_accel.conf"
west flash

Building and Running for reel board¶

The reel board is equipped with MMA8652FC accelerometer. Sample can be built and executed for the reel board as follows:

west build -b reel_board samples/sensor/fxos8700 -- -DCONF_FILE="prj_accel.conf"
west flash

Building and Running for MIMXRT685-EVK¶

MIMXRT685-EVK is equipped with FXOS8700CQ accelerometer and magnetometer. Sample can be built and executed for the MIMXRT685-EVK as follows:

west build -b mimxrt685_evk_cm33 samples/sensor/fxos8700
west flash
