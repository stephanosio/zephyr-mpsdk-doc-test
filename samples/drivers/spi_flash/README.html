JEDEC SPI-NOR Sample¶

Overview¶

This sample demonstrates using the flash API on a SPI NOR serial flash memory device. While trivial it is an example of direct access and allows confirmation that the flash is working and that automatic power savings is correctly implemented.

Building and Running¶

The application will build only for a target that has a devicetree entry with jedec,spi-nor as a compatible.

west build -b nrf52840dk_nrf52840 samples/drivers/spi_flash
west flash
