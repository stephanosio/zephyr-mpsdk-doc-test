Macro to generate pinmux int using port, pin number and mode arguments This is inspired from Linux equivalent st,stm32f429-pinctrl binding.

Pin configuration configuration bit field.

Fields:

  • mode [ 0 : 4 ]
  • line [ 5 : 8 ]
  • port [ 9 : 12 ]
Parameters
portPort ('A'..'K')
linePin (0..15)
modeMode (ANALOG, GPIO_IN, ALTERNATE).

PIN configuration bitfield.

Pin configuration is coded with the following fields Alternate Functions [ 0 : 3 ] GPIO Mode [ 4 : 5 ] GPIO Output type [ 6 ] GPIO Speed [ 7 : 8 ] GPIO PUPD config [ 9 : 10 ]

Applicable to STM32F3, STM32F4, STM32L4 series
