#define STM32_REMAP_REG_MASK   0x1U

STM32F1 Remap configuration bit field.

  • reg (0/1) [ 0 : 0 ]
  • shift (0..31) [ 1 : 5 ]
  • mask (0x1, 0x3) [ 6 : 7 ]
  • val (0..3) [ 8 : 9 ]
Parameters
regAFIO_MAPRx register (MAPR, MAPR2).
shiftPosition within AFIO_MAPRx.
maskMask for the AFIO_MAPRx field.
valRemap value (0, 1, 2 or 3).
