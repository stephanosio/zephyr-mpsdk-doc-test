◆ gdb_mem_region_array

const struct gdb_mem_region gdb_mem_region_array[]
extern

Memory region descriptions used for GDB memory access.

This array specifies which region of memory GDB can access with read/write attribites. This is used to restrict memory read/write in GDB stub to memory that can be legally accessed without resulting in memory faults.
